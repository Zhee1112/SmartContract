01 — Konteks Penelitian
Apa yang Diteliti
Empat arsitektur bridge blockchain dibandingkan untuk mengukur efisiensi gas dan tingkat keamanan terhadap serangan reentrancy dan MEV sandwich.
Tier A — Baseline
Unoptimized
Tidak ada mekanisme pertahanan. Update state setelah transfer (CEI violation). Digunakan sebagai baseline pengukuran.
Tier B — Statis
Static Only
CEI pattern, struct packing (1 slot 32B), custom errors, immutable admin. Tanpa proteksi dinamis on-chain.
Tier C — Dinamis (Full)
EIP-1153 + EWS
Transient storage via MonitorMock, Early Warning System, deteksi MEV sandwich, penalty dinamis. Overhead: ~391% gas vs Tier B. 55.7x lebih mahal dari Tier D.
Tier D — Dinamis (Ringan)
Inline EIP-1153
Semua keamanan Tier C di-inline tanpa external call. Single-slot MEV, penalty inline. Overhead: 8.7% gas vs Tier B. Best cost-effectiveness: 220.1 SPG.
Bridge adalah mekanisme penghubung aset antar jaringan blockchain. Kerentanan bridge telah menyebabkan kerugian lebih dari $2 miliar dalam 3 tahun terakhir (Ronin $620M, Wormhole $320M, Nomad $190M). Penelitian ini mengajukan pendekatan defensif on-chain menggunakan fitur terbaru Ethereum.

09 — Kesimpulan
Temuan Penelitian

Ringkasan Hasil

  • Modifikasi EIP-1153 Multiguna: Dari 1 fungsi (reentrancy guard, 200 gas) menjadi 5 fungsi keamanan (reentrancy + MEV + penalty + pause + block tracking, 9,900 gas) — 48.5x lebih murah dari Tier C
  • Tier D (Rollup Ringan): Menghemat 52.9% — 88.4% gas dibanding Tier C dengan 0 external calls melalui inline assembly
  • Tier C (Rollup Full): Overhead 327% karena 5-6 external calls ke MonitorMock per transaksi
  • Cost-effectiveness terbaik: Tier D mencapai 220.1 SPG — 3.4x lebih efisien dari Tier C
  • 215/215 pengujian lulus mencakup 13 test suites: fuzz, invariant, edge case, MEV simulation, economic deterrence, dan tier comparison
  • Hasil terbukti secara statistik melalui Welch's t-test (p = 0) dan Cohen's d = 220,64 (effect size LARGE)
  • Emergency Pause tersedia di Tier C & D sebagai mitigasi aktif jika kerentanan baru ditemukan
  • Penghematan real-world: Tier D menghemat $79,000 — $213,000/bulan untuk bridge 100K tx dibanding Tier C
